## ▶️ Como visualizar o projeto

Este projeto é composto apenas por arquivos **HTML, CSS e JavaScript**, portanto não é necessário instalar nenhuma dependência.

### Opção 1 (Recomendado)

1. Baixe ou clone este repositório.

2. Abra a pasta do projeto.

3. Dê um duplo clique no arquivo `index.html`.

O documento será aberto automaticamente no navegador padrão.

---

### Opção 2 (Visual Studio Code)

Caso utilize o VS Code:

1. Abra a pasta do projeto.

2. Instale a extensão **Live Server**.

3. Clique com o botão direito em `index.html`.

4. Selecione **Open with Live Server**.

Isso permitirá visualizar as alterações em tempo real enquanto edita o documento.

---

### Estrutura esperada

```

projeto/

│

├── index.html

├── style.css

├── script.js

└── assets/

```

> **Importante:** mantenha todos os arquivos na mesma estrutura de pastas para que imagens, estilos e scripts sejam carregados corretamente.



# 📄 Template de Documentação Técnica - WeeUp

Template desenvolvido em **HTML, CSS e JavaScript** para criação de documentações técnicas de projetos, especificações funcionais e mapeamentos de processos.

O objetivo é disponibilizar um modelo moderno, responsivo e de fácil manutenção, que possa ser reutilizado em diferentes projetos da WeeUp.

---

## ✨ Funcionalidades

- 🎨 Layout moderno seguindo a identidade visual da WeeUp
- 📑 Índice lateral com navegação automática
- 🔍 Busca em tempo real no índice
- 📍 Destaque automático da seção atual (Scroll Spy)
- ⬆️ Botão "Voltar ao topo"
- 📱 Layout responsivo para dispositivos móveis
- 🖨️ Estilos otimizados para impressão/PDF
- 🧩 Componentes reutilizáveis para documentação técnica

## 🎨 Personalização

As principais cores do projeto estão centralizadas no arquivo **style.css** através de variáveis CSS.

```css
:root {

    --verde: #15F5BA;
    --azul: #4F3CC2;
    --azul-escuro: #3B2D91;

}
```

Isso permite alterar rapidamente toda a identidade visual do template.
